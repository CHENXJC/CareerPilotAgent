# CAREER-001 MVP Scope

## Included

- Deterministic local job-description analysis
- Explainable candidate-to-job fit scoring
- Honest resume keyword, bullet, and portfolio suggestions
- Editable cover-letter draft generation
- Local CSV application tracker
- Markdown analysis and CSV tracker exports
- Safe public demo profile and three synthetic job descriptions
- Read-only Demo Profile and an optional local private profile editor
- Active profile selection across scoring, resume suggestions, cover letters, and reports

## Not Included

- Automatic job applications
- LinkedIn, Seek, Indeed, or other job-board scraping
- Email sending
- Storage of a real resume or personal contact details
- Cloud sync, authentication, or sharing of private profile data
- OpenAI API or other external API integration
- Browser automation

The MVP is a planning assistant. It does not make hiring decisions and does not submit applications.

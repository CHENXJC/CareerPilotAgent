# Screenshot Guide

Eight screenshots were captured locally for CAREER-005 after completing the Demo Profile privacy review. The files listed below now exist in `portfolio/showcase_screenshots/` and are referenced by README.

## Public Screenshot Safety

Use **Demo Profile only**. Do not capture a real private profile, resume, phone number, email address, application note, tracker record, exact address, identity number, or employer correspondence.

Before every capture:

1. Select Demo Profile.
2. Use a synthetic sample job description.
3. Keep the tracker empty or use clearly synthetic data.
4. Check the visible browser area for notifications, local paths, or private tabs.
5. Save only approved captures to `portfolio/showcase_screenshots/`.
